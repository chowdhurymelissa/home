Game Modes

Competitive play is when you strive to win in order to rank up in a 5v5 match.

As you practice your aim, Death Match is one man for himself. Everyone is fighting against one another

Spike Rush is a short game with special abilities hidden around the map

Swift Play is a short game where the team to win 5 rounds first wins.

Escalation is a game mode in which players cycle through various weapons and abilities

In Agent Select, the team votes for an agent, and the game chooses a random agent from the votes; the entire team then plays the same agent in the game

All players have access to a snowball launcher weapon; the team with 50 kills first wins

Unranked games are played in the same manner as competitive games, with the exception that there is no rank up. The first team to win 13 rounds wins the game.
